# 概要

ドールの写真と名前のリストを入力すると、ドール一覧を一枚の画像として出力するサービスを作りたい
サーバー費用がかからない用にgithub pages等のサービスを使用してサーバーを必要としない簡単なサービスとしたい

# 機能

- ドールの写真と対応する名前を複数入力すると、一枚のドール一覧画像になる
- 背景テンプレートを複数用意して選べる

# 要件

- サーバー費用がかからない用にgithub pages等のサービスを使用してサーバーを必要としないフロントエンドのみの簡単なサービスとしたい
- 作成した画像はX（旧Twitter）に投稿することを想定する
